Game series option is for playing a perpetual series of games where the score is based on cumulative points from completed games. This option is a good for beginners and intermediate players to get familiar with the 28 game play. See the How-to-Play section in Help if you are not familiar with basic game play. If this format is not challenging enough, try the Card-Count-Match.

Games can be auto advanced during bidding and card play by pressing the Fast-Forward button. This way you can skip playing an uninteresting hand at any stage during the game. However, the score will be updated based on the game outcome. See Tips-n-Tricks for this and other shortcuts.

To reset score and start a new game series, Long Press the Game-Series button in the Main Menu.
